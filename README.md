# Task Manager

A web-based task management application.


## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/projectname.git
    ```
2. Navigate to the project directory:
    ```bash
    cd projectname
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
